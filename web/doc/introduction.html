Introduction¶

The Django SaaS project is a collection of reusable Django applications that provides basic functionality for a subscription-driven website.

Objectives¶

Django SaaS aims to provide building blocks that can be used to accelerate the implementation of software-as-a-service Web applications.

The application capabilities can be leveraged and customized to meet business needs such as service subscriptions, service level enforcement, service access, security, billing and business metrics.

Planned¶

  • Customer accounts with data separation.
  • Service subscription.
  • Subdomains for accounts (customer.service.com).
  • Multiple users for customer account.
  • Role-based security.
  • Integration with billing application.
